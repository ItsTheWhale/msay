# msay
minimalistic say  

# usage  

Requires Deno
`deno run <file.js> [args]`

If non-empty arguments, will join arguments with a space. Otherwise, will read STDIN until EOF.  

```
 ______________
/ Hello there! \
\   I'm Whale  /
 --------------
     \ /     
  ____|____   ____
 /         \  \  /
|  ^        \_/ /
 \_____________/
```
